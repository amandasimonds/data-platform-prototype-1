The icons below have been uploaded to the Image S3 bucket and also added to EPD Pattern Library Icon Registry.

If you have the EPD Pattern Library Icon Registry installed on your MFE, you can use the names of these icons with mat icon, like in this example below:

<mat-icon svgIcon="yourIconName"></mat-icon>

If you do not have the EPD PL Icon Registry installed and want to do so, check out the directions on the Getting Started page.

If you do not see your icon and cannot install the EPD PL Icon Registry, feel free to check if its in the S3 bucket using the Bucket Icons List Storybook Page and use the url in your own MFE's icon registry.

name: {{ icon.name }}
