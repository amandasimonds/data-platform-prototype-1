# Action Bars

Action Bars are flexible and have 2 sections within them - left and right.<br /> 

Use `<div left></div>` and `<div right></div>` inside `<uvx-action-bar></uvx-action-bar>` to distinguish where your content will be.<br />

For spacing, please use margin utility classes such as `margin-right-sm`, `margin-left-md` as you see fit.
