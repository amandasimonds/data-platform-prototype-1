# UV Grid System

The grid system enables rows and columns to be defined for layout. 
This is useful for controlling the organization of items as well as a responsive layout. 
- The grid layout by default is a 12 column grid. 
- The default vertical gap between rows can be implemented by the utility class `row-gap`
- Start with the container class `uv-container` or `uv-container-fluid`. `uv-container` will keep items next to each other horizontally. `uv-container-fluid` will stack rows.
