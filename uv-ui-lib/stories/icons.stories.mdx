# Icons

Icons are used through the MatIconRegistry. Icons used by UV are loaded in through the CDN.
- The syntax is `<mat-icon svgIcon="<icon name>"></mat-icon>`
- Icons can come in 2 colors - gray or primary. To apply a color, apply the utility class `primary` or `gray`. For example: `<mat-icon class="primary" svgIcon="back"></mat-icon>`
