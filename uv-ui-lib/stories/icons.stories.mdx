# Icons

Icons are loaded through a CDN, configured by MatIconRegistry and RegisterIconModule. You must wrap components using the IconRegistry with uvxRegistericon provider in order for icons to load.<br /> 

Syntax: `<uvx-icon icon="<icon name>"></uvx-icon>` <br /> 

Color options: `black`, `primary`, or `gray`. To use a color, apply the color property: `<uvx-icon icon="<icon name>" color="primary"></uvx-icon>`. <br /> 
*Optional*. Default is `black`. <br />  

Size options: `xs`, `sm`, or `md`. To use a size, apply the size property: `<uvx-icon icon="<icon name>" size="md"></uvx-icon>`. <br /> 
*Optional*. Default is `sm`. <br />
